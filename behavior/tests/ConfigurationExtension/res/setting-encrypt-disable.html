Pre-condition:

Confirm the widget application is not encrypted.

Test Step:

  1. Click the "Install" button to install the widget.
  2. Use the command: "app_launcher -l" in "app" user to get the app id of setting-encrypt-disable.
  3. Use the command: "vi /home/TIZEN_USER/apps_rw/xwalk/applications/'app-id'/index.html" to show the content of index.html in terminal.
  4. Click the "Uninstall" button to uninstall the widget.

Expected Result:

Test passes if the content of "index.html" is a normal text.
