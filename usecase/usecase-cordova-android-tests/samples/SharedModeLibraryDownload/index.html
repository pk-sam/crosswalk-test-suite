Note

  1. It's only for Cordova version 3.6

Precondition

  1. Put XWalkRuntimeLib.apk in a server and make sure the test device can access it using [http://host/XWalkRuntimeLib.apk]
  2. Make sure the test device has not install XWalkRuntimeLib.apk

Test Steps

  1. Download crosswalk-cordova-<version>-arm.zip and unzip it
  2. Create shared mode test suite using --xwalk-apk-url parameter:
    $ cd /localpath/crosswalk-cordova-<version>-arm
    $ ./bin/create SharedModeLibraryDownload com.example.sharedModeLibraryDownload SharedModeLibraryDownload --xwalk-apk-url=[http://host/XWalkRuntimeLib.apk] --xwalk-shared-library
  3. Build Cordova example app:
    $ cd SharedModeLibraryDownload
    $ ./cordova/build
  4. Check the apk file generated in SharedModeLibraryDownload/bin directory
  5. Install and launch the app
  6. Click "Get Crosswalk" button, check the screen
  7. Click "Install", then check the screen
  8. Click "Done", then check the screen

Expected Results:

  1. Create test suite(SharedModeLibraryDownload) successfully
  2. Build the apk successfully
  3. The app can be installed successfully
  4. The app can launch successfully
  5. Display message "Please get Crosswalk Runtime Library"
  6. Start to download the XWalkRuntimeLib.apk
  7. Display "App installed"
  8. The app can run successfully
  9. The app can exit successfully
