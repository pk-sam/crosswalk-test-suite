Test Steps

  1. Run the following commands:
    $ unzip crosswalk-cordova-<version>-arm.zip
    $ cd /localpath/crosswalk-cordova-<version>-arm
    $ ./bin/create AppSwitch com.example.appswitch AppSwitch
    $ cd AppSwitch
    $ ./cordova/build
    $ ./cordova/run
  2. Return to the homescreen
  3. Launch the web app named 'Webappswitch' from the homescreen
  4. Go to the task manager by clicking the task manager button, then re-open the app

Purpose:

Verifies that app can launch and switch from task manager successfully

Expected Result:

  1. Launch the app
  2. Return to the homescreen successfully, no white layer overlay appears between the animations
  3. Launch the app without fullscreen mode, no white layer overlay appears between the animations
  4. Re-open the app without fullscreen mode
