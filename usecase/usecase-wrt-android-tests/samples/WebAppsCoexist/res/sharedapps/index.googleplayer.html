Pre-condition

  1. The test device doesn't install the Crosswalk Runtime Library

Test Steps

  1. Install the apk named like "shared1" on test device and launch it
  2. Install the apk named like "shared2" on test device and launch it
  3. Click "Get Crosswalk" in app "shared2", then click "Done" to complete installation process
  4. Check the display content of the launched app "shared"
  5. Switch to the app named like "shared1", check the launching screen
  6. Install the apk named like "shared3" on test device and launch it
  • Note: In formal test process, when click "Get Crosswalk" button, it should launch a Google Play or third party app store on Android, or even launch an app store selector (if there are two or more app stores were installed on device) for user to select one to download Crosswalk Runtime Library
