Pre-condition:

Immersive fullscreen mode:

  1. It can hide the virtual navigation bar (back, home, menu) with Full-screen Immersive Mode
  2. When slide from top of status bar, it can exit full screen
  3. Immersive mode only works on Android Kitkat 4.4 or higher devices

Test steps:

  1. Install the webapp named contains'LaunchFullscreenTrueTests'(not case-insensitive)
  2. Click app icon to launch the app
  3. Click the buttons in the launched app
  4. Repeat the above three steps, install the app named contains 'LaunchImmersiveFullscreen'(not case-insensitive)
  5. Switch among the two apps during fullscreen mode by task manager
  6. Click the buttons in the launched app

Expected Output:

  1. Webapp installed successfully, no error, no warning message
  2. The launched app in immersive fullscreen mode and fullscreen mode
  3. The buttons can work well
  4. The app installed launched successfully in immersive fullscreen mode and fullscreen mode
  5. The apps switched successfully when the apps during fullscreen mode
  6. The buttons can work well
