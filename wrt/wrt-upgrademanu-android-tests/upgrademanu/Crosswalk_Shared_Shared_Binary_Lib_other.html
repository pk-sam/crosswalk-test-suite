Pre-condition

  • Pack test app with lower version (than current) of Crosswalk binary at first
  • Install the XWalkRuntimeLib.apk (lower crosswalk version than current, can be same to step 1) on test device

Test Steps

  1. Install the app named like "example_shared.apk" and launch it, then input value and click "Set localStorage", "Set IndexDB" and "Set Web SQL Database"
  2. Install the app "example_ssbl.apk" build with current Crosswalk binary and runtime lib (use "adb install -r *apk" to force upgrading), then click "Get localStorage", "Get IndexDB" and "Get Web SQL Database"

Expected Results

  1. Display "Save localStorage/IndexDB/Web SQL Database value successfully: " add input content in the box
  2. Display "Get localStorage/IndexDB/Web SQL Database value: " add input content in the box
